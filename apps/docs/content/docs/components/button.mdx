## Button Style Slots

| **Slot Name** | **Targeted Component** | **Purpose**                        |
| ------------- | ---------------------- | ---------------------------------- |
| `base`        | `Button`               | Styles the button component.       |
| `icon`        | `Button`               | Styles the icon inside the button. |
