## What is Mijn-UI?

Mijn-UI is built for developers who want more flexibility and control over their UI components. It provides:

- **Styled and Unstyled Variants**: Choose between ready-to-use styled components or unstyled primitives that allow full customization.
    
- **Extensive Variant Options**: Modify sizes, colors, and interactive states to seamlessly fit your design system.
    
- **Effortless Integration**: Install Mijn-UI via npm and start using components instantly.
